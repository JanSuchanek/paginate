# Octopress Paginate

A simple paginator for Jekyll sites.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'octopress-paginate'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install octopress-paginate
